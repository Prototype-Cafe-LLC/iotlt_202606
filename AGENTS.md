- 「Lチカ止まりじゃなく、実際にモノを動かす」体験

## 使うボード

- ツマミ（可変抵抗）を回すとSG90サーボが動き、楊枝＋旗が左右に動く
- 角度に合わせて基板の赤 LED（USER_LED_R / GPIO17）の明るさが変わる（オプション）
- 完成後はUSB充電器などで**standalone動作**可能

## Seeed XIAO RP2040 配線図（ハンズオン試作用）

### 全体配線概要

### 共通 GND は「1本＋ブレッドボード」で足りる

サーボとボリュームの両方から **GND 線が来るように見える**が、基板側では **同じ電位（共通
GND）を1つにまとめればよい**。

1. **XIAO の GND ピンを1つ**だけブレッドボードの **GND（マイナス）レール**へ接続する。
2. サーボの **茶（GND）** と、ボリューム端子の **GND 側**は、両方ともその **GND レール**に
   つなぐ。

**GPIO（例: D1 / GPIO27）を「GND 出力」として使わない。** サーボなどの電流やノイズに対して GPIO
 は設計されていないため、異常電流や誤動作・故障の原因になる。GND は **XIAO の GND ピンおよび
共通レールのみ** で取ること。

電源規格についてはボード共通の説明がある。  
参考文献: [XIAO RP2040 Wiki — Pin Map／Overview](https://wiki.seeedstudio.com/XIAO-RP2040/)

### 使用ピン

- **サーボ信号** → **D4 / GPIO6**（PWM3 A）。**I2C は使わない**（D4 のデフォルト SDA をサーボに転用）
- **ボリューム（ADC）** → GPIO26（D0）
- **LED調光** → 基板 **USER_LED_R（GPIO17）**（PWM0 B、ハード PWM、キャリア約 **1 kHz**）
- 電源はUSB給電（5V）

ピンラベルは [XIAO RP2040 Wiki の Pin Map](https://wiki.seeedstudio.com/XIAO-RP2040/) に準拠。

### サーボの線色（キット付属の目安）

サーボ付属の配線図に従う。**本キットで示される色の一例**は次のとおり。

![サーボ線色: 橙=PWM, 赤=Vcc, 茶=GND](docs/servo-wiring.png)

| 線色（例） | 役割        | XIAO 接続先 |
| ----- | ---------- | -------- |
| **橙** | **PWM**（信号） | **D4 / GPIO6** |
| **赤** | **Vcc（＋）**  | **5V**     |
| **茶** | **GND（－）** | **GND**（サーボと共通） |

図中の **0.5 or 2.4 ms** はパルス幅の目安（可動域の端寄り）。ファームでは **約 1〜2 ms／50 Hz**
で制御している。

- 流通品で **茶・橙・黄の3本のみ**（赤線が無い）タイプもある。並びが違うことがあるので、その
  場合は**販売元の表**を優先する。  
  一般向け参考: [SparkFun: Hobby Servo Tutorial](https://learn.sparkfun.com/tutorials/hobby-servo-tutorial/servo-motor-background)

### 詳細配線

| 部品           | XIAO RP2040 側   | 接続先             | 備考        |
| ------------ | --------------- | ----------------- | --------- |
| **SG90 サーボ** | 5V              | **赤**（Vcc）       | 電源        |
|              | GND             | **茶**（GND）       | 共通GND     |
|              | **D4 / GPIO6**  | **橙**（PWM 信号）    | PWM3・50 Hz |
| **ボリューム**    | 3V3             | 端子1（片側）         | -         |
|              | **GPIO26** (D0) | 端子2（真ん中ワイパー）    | ADC入力     |
|              | GND             | 端子3（もう片側）       | -         |
| **LED調光**    | GPIO17（USER_LED_R・基板） | —               | PWM0 B、約1 kHz |
